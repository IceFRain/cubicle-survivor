# 🚀 核心功能

## 🍅 1. 番茄时钟
自定时间：自由配置工作时长（默认 25 min）与休息时长（默认 5 min）

便携控制：悬浮窗/托盘图标实时显示进度，并可以通过悬浮窗/托盘图标右键菜单一键开始、暂停与继续，让节奏尽在掌握

## 💧 2. 饮水记录
目标设定：每日饮水目标（如 1800 ml），设定专属的单杯容量（如 500 ml）

便捷打卡：悬浮窗/托盘图标支持“记一杯”按照设定的单杯容量累加，直观的进度条同步显示当前饮水量与目标差距


## ⚙️ 4. 贴心配置
开机自启：一键配置，开机即刻开启健康打工的一天

悬浮窗自由切换：主窗口提供独立勾选框，可分别控制番茄钟与饮水记录的悬浮窗显示，只显示你需要的内容

# 使用说明

## 下载软件
打开链接，下载最新版本的软件

[官网链接](https://github.com/IceFRain/cubicle-survivor/releases)
版本选择：win11以上选x64版本，较老的电脑选x86版本

## 使用软件
解压下载的压缩包，双击打开“cubicle_survivor”
<br /><img width="217" height="121" alt="image" src="https://github.com/user-attachments/assets/ea3a13bb-f528-49e1-b331-4836a92662f7" />

软件默认以悬浮窗状态运行在屏幕正中间，双击悬浮窗或托盘图标即可打开主界面进行配置，建议勾选“开机自启”，下次开机时即可自动启动
